Unit
    Wir brauchen als Hilfsmittel noch einen weiteren Begriff, und zwar, dass eine Zahl _beliebig, aber dann fest_ (BAF)
    ist.
    Damit ist gemeint, dass es jede Zahl sein könnte ("beliebig"), man also keine Annahmen treffen kann, um welche
    Zahl es sich handelt.
    Andererseits geht es aber um eine einzige Zahl, die sich nicht mitten in der Überlegung ändert ("fest").
    -
    Beispiel:
        Sei n€N _beliebig, aber dann fest_ (BAF). Dann ist entweder n oder n+1 eine gerade Zahl.
    n ist beliebig, also kann man nicht wissen, ob n gerade ist -- n kann gerade oder ungerade sein.
    n ist dann aber fest. Man kann also in der weiteren Betrachtung, z.B. bei der Untersuchung von n+1, davon ausgehen,
    dass es sich immer noch um das selbe n handelt.
    -
    n ist _beliebig_. n kann also gerade oder ungerade sein.
    Wenn n gerade ist, dann stimmt die Aussage, dass entweder n oder n+1 gerade ist.
    Wenn n ungerade ist, dann ist n+1 gerade. Das lässt sich sagen, weil n _fest_ ist -- n+1 ist gerade, weil das
        n darin immer noch das selbe n ist, das jetzt ungerade ist.
